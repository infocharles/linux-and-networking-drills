# Review Guide

## Start here
1. Read the top-level README.
2. Check week-03/evidence/github-access.txt.
3. Check week-03/evidence/server-copy-proof.txt.
4. Open the imported Week 1 and Week 2 folders.
5. Review the commit history and pull request proof.

## What to pay attention to
- Whether the imported folders are real and complete
- Whether the repository layout is easy to understand
- Whether the branch history is clean
- Whether the pull request summary clearly explains the work
