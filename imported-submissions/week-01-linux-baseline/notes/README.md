# Week 1 Linux Baseline Submission

## Overview
This folder contains a baseline inspection and setup on a shared Ubuntu server. The work includes system inspection, access verification, permission management, a health-check script, and scheduled automation.

## Location
Submission directory:

/week01-submissions/charles-linux-baseline/

## Folder Structure
- `notes/`  Documentation and handover notes
- `scripts/`  Health-check script
- `evidence/` Command outputs and proof of work

## Key Files
- `scripts/health-check.sh` Script used to inspect system health
- `evidence/system-info.txt` Host identity and system details
- `evidence/users-and-groups.txt` User and access context
- `evidence/permissions-proof.txt` Ownership and permission setup
- `evidence/services-proof.txt` Service and system inspection
- `evidence/cron-proof.txt` Cron job configuration and proof
- `evidence/health-check-output.txt` Output from automated runs

## Summary
The submission demonstrates safe working practices on a shared server, structured evidence collection, and basic operational automation using cron.
